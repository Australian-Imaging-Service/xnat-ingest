*XNAT Ingest* is a toolkit for capturing data from instruments and uploading it to XNAT.


Installation
------------

*XNAT Ingest* can be installed for Python >=3.11 using *pip*

.. code-block:: console

    $ python3 -m pip install xnat-ingest
